    # p-value

    Un altro metodo per verificare se √® corretto accettare o rifiutare una ipotesi √® il calcolo del p-value, noto anche come significativit√† osservata. In parole semplici, potrebbe capitare di osservare dei dati che inducono al rifiuto dell’ipotesi nulla $H_0$. Il p-value aiuta a capire se la differenza tra i dati osservati e l’ipotesi nulla sia dovuta a un puro caso, oppure se in realt√† tale differenza √® statisticamente rilevante, ovvero difficilmente dovuta a un puro caso.

    Consideriamo un campione $X_1, …, X_n$ preso da una popolazione della quale non conosciamo le media $\mu$, e definiamo l’ipotesi nulla $$H_0: \mu \leq \mu_0$$ Normaliziamo i dati tramite un semplice z-test, definendo il valore osservato $$T = \frac{\overline{X} - \mu_0}{\sigma/\sqrt{n}}$$ che avr√† una distribuzione normale standardizzata $N(0,1)$.

    Il p-value √® definita come la probabilit√† di ottenere un risultato “pi√π estremo” di quello osservato $T$, sotto l’ipotesi che $H_0$ sia vera.

    Definiamo cosa si intende per “pi√π estremo”.

    Nel nostro esempio ($H_0: \mu \leq \mu_0$) abbiamo un test unilaterale destro, ovvero il z-test rifiuta $H_0$ se $T > c$ per un certo $c$ fissato. Perci√≤ per valori estremi si intedono dati osservati $T$ che vanno verso la coda destra di una normale. Perci√≤, il p-value in questo caso √® definito come $$p(X_1,…,X_n) = P(Z \geq T ;\vert; H_0) = 1 - \Phi(T) = \Phi(-T)$$ dove $Z \sim N(0,1)$.

    pi√π il p-value √® piccolo, pi√π la significativit√† √® alta poich√© il risultato del test ci dice che $H_0$ non spiega adeguatamente i dati osservati.

    Perci√≤ il test del p-value consiste nel $$\text{rifiutare } H_0 \iff p(X_1, …, X_n) < \alpha$$
